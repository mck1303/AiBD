To jest projekt poświęcony badaniom nad ilością alkoholu spożywaną przez mieszkańców poszczególnych państw ONZ.

W folderze OriginalData można znaleźć pierwotne zestawienie.

W folderze CommandFiles znajduje się program napisany w języku Python, któy umożliwa formatowanie zestawienia.

W folderze Analysis Data zapisany zostało finalne zestawienie, które jest zgodne z zasadami Tidy Data. Znajdziemy w nim również folder Metadata wraz z przewodnikiem.
